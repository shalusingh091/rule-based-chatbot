
# 🤖 AI Rule-Based Chatbot

## 📌 Project Description
This is a simple AI-style Rule-Based Chatbot built using Python.
It responds to user inputs based on predefined rules using:
- if-elif conditions
- Regular Expressions (pattern matching)

## 🚀 Features
- Greeting responses
- Name response
- Time response
- Help command
- Exit command

## ▶️ How to Run
1. Install Python
2. Open terminal in this folder
3. Run:
   python main.py

## 📚 Concepts Used
- Functions
- Loops
- Conditional Statements
- Regular Expressions
- Basic NLP idea (keyword matching)
